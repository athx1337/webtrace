## Overview

TRACE is a Domain Intelligence API system comprising a React-based frontend dashboard and a FastAPI backend. The frontend provides a visual OP_CENTER terminal interface, while the backend orchestrates concurrent domain analysis (URL validation, DNS lookups, whois info, and various engine checks) to compute a trust score and generate an AI-driven explanation.

## Data Flow

1. User submits domain to `/api/analyze`.
2. Backend normalizes the URL.
3. Backend concurrently fetches DNS, Domain info, URL heuristics, and Engine results.
4. Trust Score calculation logic processes the aggregated responses synchronously.
5. AI explanation module takes the aggregated result and generates a summary via Gemini.
